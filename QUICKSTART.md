# ⚡ Quick Start

## Play NOW (10 seconds)

```bash
python3 zagreus_dungeon.py
```

## Tips

1. Read descriptions carefully
2. Option 1 is NOT always best
3. Don't be greedy (creatures wake up!)
4. Checkpoints save at key moments
5. Learn from deaths

## First Choices

Turn 1: Search water (option 1)  
Turn 2: Search corpse (option 1)  
Turn 3: Take tinderbox + coins (option 2) ← NOT option 1!  

## When You Die

1. Read the lesson
2. Choose "Load latest checkpoint"
3. Try different approach

## That's It!

No installation. No setup. Just play!
